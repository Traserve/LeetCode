#### [剑指 Offer 05. 替换空格](https://leetcode-cn.com/problems/ti-huan-kong-ge-lcof/)

请实现一个函数，把字符串 `s` 中的每个空格替换成"%20"。

 

**示例 1：**

```
输入：s = "We are happy."
输出："We%20are%20happy."
```

 

**限制：**

```
0 <= s 的长度 <= 10000
```

## 思路

先遍历一遍字符串，确定空格的数量。这样就可以计算出结果字符数组的长度。然后从后向前遍历把每个字符串填入数组中，遇到空格则填充%20
